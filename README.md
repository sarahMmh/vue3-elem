# vue3-elem

> 为了练习vue3以及ts而创建的项目

## 项目运行

You can replace `yarn` with `npm run` here.

```bash
# build for production
yarn build

# development mode
yarn dev

# run unit tests
yarn test

# serve the bundled dist folder in production mode
yarn serve
```

## 技术栈

vue3 + vuex + webpack + typescript
